## 🔑 How to Verify Student Status

Most deals use one of these three verification methods:

**1. 📦 GitHub Student Developer Pack** *(recommended — unlocks 30+ deals in one application)*
> Apply at [education.github.com/pack](https://education.github.com/pack) — upload a photo of your student ID or use your `.edu` email. Approval typically takes 1–7 days.

**2. ✅ SheerID** *(used by Adobe, Spotify, Perplexity, and others)*
> Instant verification at point of signup — enter your `.edu` email or upload your enrollment document at [sheerid.com](https://www.sheerid.com/). Usually instant.

**3. 📧 Your institution's `.edu` email address**
> JetBrains, Figma, Microsoft Office 365, and Google One all verify automatically with a valid `.edu` email — no extra steps needed.

## 🚀 Deploy to GitHub Pages (60 seconds)

```bash
# 1. Fork or clone this repo
git clone https://github.com/yourusername/student-deals.git
cd student-deals

# 2. (Optional) customise the deals list in index.html

# 3. Push to GitHub
git add .
git commit -m "init"
git push origin main
```

Then in your GitHub repo:

1. Go to **Settings → Pages**
2. Set source to **Deploy from branch → main → / (root)**
3. Click **Save**

Your site will be live at `https://yourusername.github.io/student-deals` within a minute.

## 🤝 Contributing

Found a deal that's missing or outdated? Submit a pull request!

### Adding a new deal

Open `index.html` and add an entry to the `DEALS` array in the `<script>` section:

```javascript
{
  name: "Tool Name",
  cat: "Category",           // Development | Design | AI | Cloud | Learning | Productivity | Communication | Entertainment | Professional | Game Dev
  discount: "FREE",          // e.g. "FREE", "50% OFF", "$100 credits", "FREE 6 months"
  desc: "What the tool does and why students should care — one to two sentences.",
  req: "What the student needs to verify and claim the deal",
  note: "Optional tip, caveat, or extra context",
  tags: ["category", "free"],  // free | discounted | credits | development | design | ai | cloud | learning | productivity | communication | entertainment | professional | game
  pack: false,               // true if available via the GitHub Student Developer Pack
  link: "https://official-student-signup-link.com"
}
```

**Guidelines:**
- Only add deals you have personally verified are still active
- Link directly to the student/education signup page — not the homepage
- Keep descriptions concise and focused on student value
- Set `pack: true` for anything claimable through the GitHub Student Developer Pack
- Also add the deal to the correct section in `README.md`

### Steps

```bash
git fork https://github.com/yourusername/student-deals
git checkout -b add-toolname-deal
# Edit index.html (add to DEALS array)
# Edit README.md (add to the correct category table)
git commit -m "add: ToolName student deal"
git push origin add-toolname-deal
# Open a pull request on GitHub
```

## 📌 Tips for Students

1. **Start with the [GitHub Student Pack](https://education.github.com/pack)** — it unlocks 30+ deals in one application and is by far the most valuable single step
2. **Use your `.edu` email everywhere** — services like JetBrains, Figma, and Microsoft verify automatically
3. **Set annual calendar reminders** — most deals require yearly re-verification of student status
4. **Check your university portal first** — your institution may already have site licences for [Office 365](https://www.microsoft.com/en-us/education/products/office), [Zoom](https://zoom.us/pricing/education), [LinkedIn Learning](https://www.linkedin.com/learning/), and [Grammarly](https://www.grammarly.com/edu) at no cost to you
